# Rote Skill

这个仓库提供了一个通过 `rote-toolkit` 操作 Rote 的通用 AI agent skill。

## 包含文件

- `SKILL.md`：skill 的触发描述与决策规则
- `references/commands.md`：命令参考、高阶 playbook 与失败处理约定
- `agents/openai.yaml`：skill 的 UI 元数据
